## Installation

```shell
git clone https://github.com/rpowel/maze
cd maze
pip install -r requirements.txt
cd app
python app.py
```

## Usage

### Gui Based

Running app.py (after installing requirements) will open a window where you can select
the size and type of maze you would like to use.
